#  Overview

The **Restaurant Reservation System** is a desktop application developed to simplify restaurant reservation and customer management.

The application enables customers to reserve tables while allowing restaurant staff to efficiently manage reservations, customer details, and restaurant operations.

The project demonstrates database connectivity, desktop application development, and backend integration using modern Java technologies.

---

#  Features

###  Reservation Management

- Create new reservations
- Update reservation details
- Cancel reservations
- View reservation history

###  Customer Management

- Register customers
- Update customer information
- Search customers
- Delete customer records

###  Table Management

- Table availability
- Reservation status
- Seat management

###  Database Operations

- Store reservation data
- Customer database management
- SQL-based persistent storage

###  User Interface

- JavaFX Desktop Application
- Responsive Forms
- Clean User Interface
- Easy Navigation

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Kanishka-Rajesh/Restaurant-Reservation-System.git
```

---

## Navigate

```bash
cd Restaurant-Reservation-System
```

---

## Create Database

Open MySQL.

Run

```
database/schema.sql
```

to create the required database and tables.

---

## Configure Database

Update the database configuration with your MySQL username and password.

---

## Run Project

Using Maven

```bash
mvn clean install
```

Run the application from your IDE or execute the generated application.
